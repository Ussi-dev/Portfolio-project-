Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

print("Hello, world!")

Python was created by Guido van Rossum and released in 1991.

  • Easy to learn
  • Extensive libraries
  • Large community support
Variables and Data Types

Variables in Python are dynamically typed.

Data types include integers, floats, strings, lists, tuples, and dictionaries.

x = 42 name = "Alice"
  • int - Integer
  • str - String
  • list - List
Control Structures

Python supports conditional and looping control structures.

Indentation is critical in Python to define blocks of code.

if x > 10: for item in items:
  • if, elif, else
  • for loops
  • while loops
Functions

Functions are defined using the def keyword.

They can have parameters and return values.

def greet(name):

Functions help in code reusability and readability.

  • Built-in functions
  • User-defined functions
  • Lambda functions
Modules and Imports

Modules allow you to organize code into separate files.

You can import existing modules using the import statement.

import math from datetime import date
  • Standard library modules
  • Third-party packages
  • Custom modules
